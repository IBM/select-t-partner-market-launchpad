# Glossary

Key terms used throughout the Partner Market Launchpad program.

---

| Term | Definition |
|------|------------|
| **Partner Pipeline** | The stages a partner progresses through: Learning → Trying → Buying → Onboarding → Getting Help |
| **Track** | A single product-focused Launchpad run, containing theory and 5 labs |
| **Cohort** | The group of ~20 attendees in a single Launchpad session |
| **Capstone** | Lab 501 — the final, comprehensive lab that integrates all prior learning |
| **BTL** | Business Technical Leader — client-side technical decision maker |
| **Facilitator** | The person running the Launchpad session |
| **Lab Environment** | The pre-provisioned cloud/product environment used for hands-on labs |
| _Add more_ | _As needed per product track_ |
